# 深海霓虹捕魚 (Neon Deep Sea Fishing) - HTML5 手遊

這是一款專為行動裝置（手遊）與網頁優化的原生 HTML5 捕魚遊戲。採用了現代化的 **Cyberpunk 霓虹深海** 視覺風格，搭配流暢的 procedural vector 游動動畫、炫麗的粒子效果與網頁內建音效合成技術。

## 🌟 遊戲特點
1. **迷幻深海霓虹視覺**：
   - 魚群採用貝茲曲線在 Canvas 上實時繪製，尾部具備流暢的正弦波擺動動畫。
   - 特製的「雙重筆觸描邊」技術，無須高負載的 blur 濾鏡，在任何低配行動裝置上都能以 60FPS 渲染高質感的霓虹發光效果。
   - 動態變幻的海底光影（Sun Rays）、擺動的海草與隨機上升的水泡。
2. **經典街機發射操作**：
   - 底部設有能量砲台，點擊畫面即可轉向射擊。子彈抵達目標後會炸開為圓形網狀能量圈，對範圍內的魚群進行捕獲判定。
3. **限時挑戰與隨機事件**：
   - 每局 60 秒限時得分，並有連擊 (Combo) 累加倍率。
   - **魚潮來襲**：大批霓虹魚群以交叉、波浪隊形快速穿過畫面。
   - **Boss 霓虹黃金龍降臨**：採用多節點連鎖跟隨算法（IK），展現平滑如蛇行般的黃金龍身軀。Boss 擁有血量條，擊破可獲得 5000 分大獎。
4. **四大特殊技能系統**：
   - **❄️ 冰凍 (Freeze)**：暫停所有魚群移動 5 秒，背景呈冰藍冰封特效。
   - **⚡ 雷射 (Laser)**：可手動朝任意垂直切面發射一道粗大的電光雷射，秒殺路徑上的普通魚。
   - **💣 炸彈 (Bomb)**：於隨機中心位置引爆超大能量波紋，炸飛周邊大範圍魚群，並伴隨螢幕震動。
   - **🔥 雙倍得分 (Double)**：開啟後 10 秒內捕獲的所有得分翻倍，畫面呈紅色閃爍霓虹框。
5. **純代碼合成音效 (Web Audio API)**：
   - 擺脫加載 MP3 檔案的慢速與跨域問題。
   - 運用振盪器 (Oscillator) 與包絡線即時合成低延遲音效：包括砲彈發射聲、清脆金幣聲、科幻雷射充能聲、低頻爆炸聲與氣泡咕嚕聲。
6. **響應式適配**：
   - 支援 PC 瀏覽器與 iOS/Android 行動端瀏覽器。
   - 使用 CSS Media Queries 與 Canvas resize 動態縮放，橫屏與豎屏均能完美自適應排版。

## 🚀 快速啟動
由於遊戲使用 Web Audio API，部分瀏覽器基於安全性限制，不允許以 `file://` (直接點擊 index.html 檔案) 播放音效，因此**推薦在本地啟動輕量伺服器**來執行：

### 方法一：使用 VS Code Live Server (最推薦)
1. 在編輯器中開啟此目錄。
2. 點擊右下角的 **Go Live** 按鈕。
3. 在瀏覽器中開啟 `http://127.0.0.1:5500` 即可暢玩。

### 方法二：使用 Node.js (npm / npx)
在終端機中，切換到本專案目錄並執行：
```bash
npx http-server ./ -p 8080
```
或使用 python：
```bash
python -m http.server 8080
```
然後在瀏覽器中造訪 `http://localhost:8080` 即可。

## 🛠 檔案結構
- `index.html`：遊戲主骨架與 HUD 面板布局。
- `style.css`：玻璃擬態 (Glassmorphism)、霓虹動畫與響應式適配樣式。
- `game.js`：核心遊戲循環 (Game Loop)、物理模擬、音效合成、Canvas 圖形渲染。
